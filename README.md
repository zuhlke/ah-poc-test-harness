# ah-poc-test-harness

Exercises the ah-poc-papi

- Average latency

- Average throughput

## Usage

Usage: `python3 main.py <option>`

- Use the option `[-n <N>]` to measure average latency over N requests
- Use the option `[-t <T>]` to measure average throughput over T seconds
